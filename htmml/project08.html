If you take a look at the HTML for this page you will notice that the section numbering that you can see below is not include in the markup

The section numbering that you can see in this page is automatically generated by CSS using the 'counter-reset' and 'counter-increment' properties.

This could be used in many scenarios, such as academic papers, technical documentation, legal documents, etc..., where section numbering is useful or needed. It enables the writer to give their attention to the content being written and not to the section numbering. It also allows the writer to add new sections without being forced to re-number all the following sections manually.
